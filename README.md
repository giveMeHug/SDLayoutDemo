# SDLayoutDemo
一个简单的图文混排，图片大小自适应，限制最大宽度，使用indexPath缓存行高。支持cell编辑插入、收藏、点赞、置顶、删除功能。
缓存行高功能，能够支持90%以上的需求，如果数据计算较为复杂，可以对数据预处理+缓存行高，就相对较好了。
